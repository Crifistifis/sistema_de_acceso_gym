# sistema_de_acceso_gym
Programa de escritorio con conexión a un lector de huellas para el contro de acceso de clientes a un establecimiento de caracter deportivo

🛠 Herramientas y Tecnologías Requeridas
Para colaborar en este proyecto, asegúrate de tener instalado y configurado el siguiente entorno:

-Lenguaje de Programación: C#. Las aplicaciones en este lenguaje corren de forma nativa y eficiente en Windows.

-Entorno de Desarrollo (IDE): Visual Studio. Es considerado el mejor entorno para construir apps de escritorio.

-Framework de Interfaz de Usuario: WinForms

-Base de Datos: Dependiendo de la configuración final del servidor del gimnasio, el sistema se conectará a SQL Server, MySQL o SQLite.
